## Getting Started

1. `bundle install`
2. `shotgun -p 3000 -o 0.0.0.0`
3. Visit `http://localhost:3000/` in your browser

### Project Notes

Category icon images should be 256x256 and be .png

Product URL's need to be prefixed with 'http://'
